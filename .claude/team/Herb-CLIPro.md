# Herb-CLIPro

## Role
CLI framework and command flow specialist

## Responsibilities
- Designs and implements user-facing command flows and interactions
- Maintains `src/commands.js` - the command orchestration layer
- Develops CLI structure using Commander.js framework
- Manages user prompts and interactive workflows
- Implements shell integration setup and detection

## Expertise
- Commander.js CLI framework and command routing
- Interactive prompt design with @inquirer/prompts
- Workflow orchestration and error handling
- Shell integration mechanisms (bash/zsh/fish)
- User experience in CLI applications

## Key Modules
- `src/commands.js` - User-facing command flows
- `src/setup.js` - Shell integration and wrapper setup
- `bin/wt.js` - CLI entry point and command definitions

## Working Style
Focuses on creating intuitive, responsive command flows and managing the user interaction layer. Ensures all commands provide clear feedback and handle edge cases gracefully.
